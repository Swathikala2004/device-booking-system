# Device Booking System

A Full Stack Web Application for managing and reserving shared devices.

## Technologies Used
Frontend: React.js, Bootstrap  
Backend: FastAPI (Python)  
Database: SQLite  

## Features
- User login
- View available devices
- Book devices
- Cancel bookings
- Admin panel to manage devices
